# Escalas Pediátricas de Evaluación Obstructiva: Westley (Crup) y TAL (Asma)

## 1. Escala de Westley para Crup (Laringotraqueobronquitis)

**Objetivo**: Evaluar gravedad del crup y diferenciar de asma (estridor vs sibilancia).  
**Fuentes**: EUNACOM, Manual de Clínicas de Chile (MINSAL).  
**Signo cardinal**: Estridor (vía aérea superior) → NO sibilancia (vía aérea inferior).

### Diferenciación clínica clave (pediatra debe dominar)

**Estridor**  
- Tono alto, monofónico o musical  
- Predominantemente **inspiratorio** (puede ser bifásico en grave)  
- Mejor audible en cuello / tráquea  
- Aumenta con agitación / llanto  
- Tos perruna, voz ronca  
- Mejora con epinefrina racémica / heliox  
- Etiología típica: crup viral, epiglotitis, cuerpo extraño

**Sibilancia**  
- Tono alto, polifónico, silbante  
- Predominantemente **espiratorio** (puede ser bifásico en asma grave)  
- Difusa en campos pulmonares  
- Asociada a taquipnea, retracciones intercostales/subcostales  
- Mejora con beta-2 agonistas (salbutamol)  
- Etiología típica: asma, bronquiolitis, aspiración

### Componentes Escala Westley (máximo 17 puntos)

| Ítem                  | Descripción                                                                 | Puntos |
|-----------------------|-----------------------------------------------------------------------------|--------|
| Estridor              | Ausente = 0<br>Audible con estetoscopio/agitación = 1<br>Audible a distancia = 2 | 0–2    |
| Retracciones          | Ninguna = 0<br>Leve (solo supraesternal) = 1<br>Moderada (intercostal + supra) = 2<br>Severa (supra + inter + subcostal + aleteo) = 3 | 0–3    |
| Entrada de aire       | Normal = 0<br>Disminuida = 1<br>Muy disminuida / silencio torácico = 2      | 0–2    |
| Cianosis              | Ausente = 0<br>Con agitación = 4<br>En reposo = 5                           | 0, 4–5 |
| Nivel de conciencia   | Normal = 0<br>Alterado (letargia, confusión) = 5                            | 0 o 5  |

**Clasificación**  
- ≤2 → Leve  
- 3–7 → Moderada  
- ≥8 → Severa (hospital / UCI probable)

## 2. Escala de TAL (Pulmonary Score / Score de Tal) para crisis obstructiva inferior (Asma / Bronquiolitis con sibilancias)

**Objetivo**: Cuantificar gravedad de obstrucción de vía aérea inferior (sibilancia = asma probable o bronquiolitis).  
**Fuentes**: Protocolos de URGENCIA.  
**Signo cardinal**: Sibilancia difusa + respuesta a broncodilatadores.

- Edad de corte para Diagnóstico: **2 años**
- Menor de 2 años = Bronquiolitis (si es 1er episodio)
- 2 años o más = Sibilancia / Asma (o sibilancia recurrente)

### Componentes Escala TAL (0–12 puntos)

| Ítem                        | Criterios                                                                 | Puntos |
|-----------------------------|---------------------------------------------------------------------------|--------|
| Frecuencia respiratoria     | <6 meses: <40=0, 41–55=1, 56–70=2, >70=3<br>>6 meses: <30=0, 31–45=1, 46–60=2, >60=3 | 0–3    |
| Sibilancias                 | Ausentes = 0<br>Terminales / solo con estetoscopio = 1<br>Toda espiración = 2<br>Inspiración + espiración audibles sin estetoscopio = 3 | 0–3    |
| Uso de músculos accesorios / Retracciones | Ausente = 0<br>+ (leve, aleteo) = 1<br>++ (intercostal) = 2<br>+++ (supra/subcostal + paradójico) = 3 | 0–3    |
| Saturación O₂ (aire ambiente) | ≥95% = 0<br>90–94% = 1<br><90% o requiere O₂ para ≥92% = 2–3 (según protocolo local) | 0–3    |

**Clasificación aproximada** (adaptada protocolos chilenos)  
- <5 → Leve → ambulatorio (salbutamol c/4–6 h)  
- 5–8 → Moderada → hospital abreviado / observación (salbutamol c/10–20 min ×5, prednisona)  
- ≥9 → Severa → hospital, O₂, salbutamol continuo ± ipratropio ± corticoides IV

### Algoritmo resumido TAL (basado en protocolos chilenos)

- <5 puntos → Salbutamol 2 puff c/4–6 h × 5–7 días. Reevaluación 24 h.  
- 6–8 puntos → Salbutamol 2 puff c/10 min × 5 dosis → evaluar 1 h. Prednisona 1–2 mg/kg/d.  
- ≥9 puntos → Oxígeno, salbutamol nebulizado continuo, corticoides IV, considerar magnesio, KNTR si no responde.
